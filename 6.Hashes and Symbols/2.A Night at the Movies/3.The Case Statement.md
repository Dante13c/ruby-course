### The Case Statement

5 min

Good work! Now we’ll want to create the main body of our program: the `case` statement, which will decide what actions to take based on what the user types in.

`if` and `else` are powerful, but we can get bogged down in `if`s and `elsif`s if we have a lot of conditions to check. Thankfully, Ruby provides us with a concise alternative: the `case` statement. The syntax looks like this:

```
case language
  when "JS"
    puts "Websites!"
  when "Python"
    puts "Science!"
  when "Ruby"
    puts "Web apps!"
  else
    puts "I don't know!"
end
```

The `else` is what the `case` statement will do if it doesn’t match any of its `when` statements to the `case` (in this case, the value of `language`).
